## About Me / 关于我

- 来自天津，现阶段主要在做学生开发者视角下的实用型项目。
- 比较关注桌面工具、全栈应用和“真的能被人用起来”的产品体验。
- 最近投入较多的方向是 `Rust`、`Tauri v2`、`Vue`、`Spring Boot`、`FastAPI` 和多 Agent 应用。
- 我希望把每个项目都做得更完整一点，不只是能跑起来，还要有清晰结构、可维护性和真实使用价值。

## Current Focus / 近期方向

```text
- 用 Rust + Tauri 打磨更现代的桌面工具
```
